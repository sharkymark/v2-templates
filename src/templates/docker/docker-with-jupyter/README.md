# JupyterLab template for a workspace in a container on a Docker host

### Apps included

1. A web-based terminal
2. JupyterLab IDE
3. VS Code Web IDE

### Requirements

1. A wildcard subdomain must be enabled (either with Coder's default tunnel or you manually configuring it)
